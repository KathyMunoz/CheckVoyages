# CheckVoyages
Projet fil rouge dans le cadre de ma formation à l'ADRAR en tant que Développeuse Web et Web Mobile

## Presentation

Check Voyages est un site qui permet d'échanger sur des destinations de voyages. La version initiale était en HTML5 / CSS3 / JS.

Cette version est une refonte en y ajoutant du PHP + MySQL.
